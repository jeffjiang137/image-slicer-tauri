# 图片长图拼接工具 Tauri 版

这是一个更轻量的桌面版方案：界面用 HTML/CSS/JavaScript，桌面壳用 Tauri，最终高清拼接保存由 Rust 后端完成。

当前版本保留了这些核心能力：

- 拖入图片或文件夹导入
- 首次导入自然排序，后续导入追加
- 左侧长图实时预览，支持滚动和缩放
- 右侧列表点击定位、拖动改顺序、Delete 删除、Ctrl+A 全选
- 间距与间距颜色设置
- 保存为 JPG 长图，并记住上一次保存目录
- 顶部提示词功能
  - 单张生成
  - 批量生成 / 停止
  - 复制全部
  - API 设置
  - 未配置 API Key 时输出统一模板提示词

## 功能简介

- 极限词排查：对图片中的文案进行 OCR 识别，并自动匹配广告法、电商平台常见风险词，快速标出可能需要修改的内容。
- 极限词库：内置常用极限词分类词库，支持查看、搜索、增删改和恢复默认，方便按店铺或平台规则维护自己的审核标准。
- 图片间距填充：支持设置图片之间的间距，并可选择纯色、延展、渐变、模糊、镜像和微阴影等填充方式，让拼接过渡更自然。
- 指定尺寸导出：可按指定宽度导出完整 JPG 长图，也可以按尺寸规则切片导出，适配详情页上传、平台分段展示和后续二次编辑。
- 生成复刻提示词：根据参考图片分析画面风格、构图、产品元素、文字排版和色彩氛围，生成可用于复刻同款视觉效果的中文提示词。
- 套版生图：以原图版式为参考，结合新的产品图和提示词生成同风格主图或详情图，适合快速产出系列化电商视觉。
- 替换套版：保留参考图的整体构图、背景、卖点布局和视觉风格，将其中的产品替换为新上传的产品素材。
- 分层套版：提供文字、矩形、图片等图层编辑能力，可以移动、缩放、对齐和调整层级，在套版基础上做更精细的二次排版。

## 环境要求

需要先安装：

- Node.js 18+
- Rust
- Windows WebView2

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run tauri dev
```

## 打包

```bash
npm run tauri build -- --no-bundle
```

也可以直接双击：

```bat
build_tauri.bat
```

脚本会自动检查 Node.js、npm、Rust、cargo，环境齐全后再安装依赖并打包。默认只生成 exe，不额外打 MSI 安装包，避免 WiX 下载超时。

## 说明

- 前端负责预览、排序、提示词交互。
- Rust 后端负责读取本地文件、递归文件夹、保存高清 JPG、读写配置。
- 配置文件会保存到当前用户目录下的应用配置目录，不写死到程序安装目录。

# 电商设计助手

电商设计助手是一款面向电商运营、视觉设计和商品上新场景的 Windows 桌面工具。它把图片长图拼接、详情页切片、广告风险词排查、提示词提取、套版生图和分层套版编辑放在一个本地应用里，帮助团队更快完成主图、详情页和批量视觉素材的生产。

项目采用 Tauri 桌面壳，前端使用 HTML/CSS/JavaScript，图片处理、文件读写、OCR、配置保存和接口调用由 Rust 后端完成。默认不内置任何 API Key，使用者需要在软件内配置自己的模型接口。

## 产品定位

这不是单纯的拼图工具，而是一个面向电商图片工作流的轻量生产台。

核心目标是减少运营和设计在以下环节的重复劳动：

- 把多张详情图按正确顺序拼成长图。
- 按平台尺寸要求导出长图或切片。
- 给图片之间补间距、补边、做自然过渡。
- 从参考图中提取可复刻的生成提示词。
- 用产品图和参考版式快速生成同风格主图或详情图。
- 对套版图层做二次排版和精细调整。
- 检查图片文案中的极限词、违禁词和广告法风险词。

## 核心功能

### 图片导入与长图拼接

- 支持拖入图片或文件夹导入。
- 支持 JPG、JPEG、PNG、WEBP、BMP、TIF、TIFF 等常见格式。
- 首次导入按文件名自然排序，后续导入自动追加。
- 左侧提供长图实时预览，可滚动、缩放和快速检查整体效果。
- 右侧图片列表支持点击定位、拖动排序、Delete 删除、Ctrl+A 全选。
- 保存时记住上一次保存目录，减少重复选择路径。

### 间距填充与补边

- 支持设置图片之间的间距。
- 支持纯色、边缘延展、渐变、模糊、镜像、微阴影等填充方式。
- 支持图片侧边、顶部和底部补边，方便适配不同平台的详情页宽度和留白要求。
- 填充效果会参与实时预览，导出结果与预览保持一致。

### 指定尺寸导出

- 支持按指定宽度导出完整 JPG 长图。
- 支持按尺寸规则切片导出，适用于详情页分段上传、平台图片尺寸限制和后续二次编辑。
- 前端负责预览和参数配置，Rust 后端负责高质量缩放、拼接、裁切和保存。

### 生成复刻提示词

- 支持对参考图进行视觉分析，提取画面风格、构图、产品元素、文案排版、色彩和光影信息。
- 可生成适合图生图、文生图和商品视觉复刻的中文提示词。
- 支持单张生成、批量生成、停止任务和复制全部结果。
- 未配置 API Key 时会输出统一模板提示词，不阻塞基础拼图功能。

### 套版生图

- 使用参考图作为版式和风格依据，结合新的产品图生成同风格电商主图或详情图。
- 支持选择常见画幅比例，例如 1:1、3:4、4:5、16:9、9:16 等。
- 适合批量生产系列化商品视觉，保持店铺图片风格一致。

### 替换套版

- 保留参考图的构图、背景、卖点布局和视觉氛围。
- 将原图中的产品替换为新上传产品素材。
- 适用于同类商品换款、换色、换材质的快速出图场景。

### 分层套版编辑

- 支持图片、文字、矩形等图层编辑。
- 支持移动、缩放、对齐、层级调整、混合模式、文字样式和描边设置。
- 支持图层历史回退，便于在套版基础上做精细排版。
- 适合在 AI 生成图之后继续做人工可控的二次设计。

### 极限词排查

- 支持对图片文字进行 OCR 识别。
- 自动匹配内置极限词、违禁词和电商广告风险词。
- 显示风险词命中情况、OCR 原文和检测状态。
- 可用于主图、详情页、活动图上线前的基础合规自查。

### 极限词库

- 内置常用风险词分类词库。
- 支持搜索、查看、添加、编辑、删除和恢复默认词库。
- 词库保存在本地，方便不同店铺或不同平台维护自己的审核标准。

### API 设置

- 支持 Gemini、OpenAI、通义千问 / Qwen、智谱 GLM 和自定义 OpenAI 兼容接口。
- 提示词提取、风险 OCR 和生图接口可分别配置。
- API Key 保存在当前用户的应用配置目录中，Windows 下使用 DPAPI 做本机用户级保护。
- 源码和安装包不包含任何真实 API Key。

## 业务逻辑

### 基础拼接流程

1. 用户导入图片或文件夹。
2. 应用读取图片元信息并按自然顺序建立图片列表。
3. 用户在列表中调整顺序、删除不需要的图片、设置间距和补边。
4. 前端实时渲染预览，保持与最终导出逻辑一致。
5. 用户选择导出模式。
6. Rust 后端按配置完成缩放、拼接、填充、切片和保存。

### 提示词与套版流程

1. 用户导入参考图或产品图。
2. 用户配置提示词 API 或生图 API。
3. 前端整理图片、提示词、画幅比例和任务参数。
4. Rust 后端调用模型接口并返回文本或生成图结果。
5. 前端展示结果、费用摘要和任务状态。
6. 用户可复制提示词、导出结果图，或进入分层编辑继续调整。

### 风险词排查流程

1. 用户对图片发起极限词检查。
2. Rust 后端调用 Windows OCR 或接口能力提取图片文字。
3. 前端使用本地风险词库进行匹配。
4. 应用展示命中的词、对应分类、OCR 文本和检测结果。
5. 用户可调整词库并重新检测。

### 配置保存逻辑

- 应用配置写入当前用户的应用配置目录，不写死到安装目录。
- 最近保存目录、间距填充模式、更新地址、API 配置等会被保存。
- API Key 保存前会做本机加密保护。
- `.env`、本地上传数据、构建产物和安装包不应提交到源码仓库。

## 环境要求

- Windows 10/11
- Node.js 18+
- Rust
- Windows WebView2 Runtime

## 本地开发

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run tauri dev
```

前端构建：

```bash
npm run build
```

生成 Windows 安装包：

```bash
npm run tauri build -- --bundles nsis
```

也可以运行：

```bat
build_tauri.bat
```

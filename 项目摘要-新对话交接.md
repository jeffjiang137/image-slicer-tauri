# 电商设计助手 / 图片长图拼接工具 Tauri 版项目摘要

## 当前版本
- 当前版本：1.0.12（本轮待发布后会继续升版）
- 版本位置：
  - `package.json`
  - `package-lock.json`
  - `src-tauri/tauri.conf.json`
  - `src-tauri/Cargo.toml`
  - `src-tauri/Cargo.lock`
  - `src/main.js` 的 `CURRENT_APP_VERSION`

## 项目定位
- Tauri 桌面版电商图片处理工具。
- 核心功能包括长图拼接、批量替换、分层套版、一键套版、提示词生成、极限词 OCR 排查、局域网更新。

## 主要模式
- 长图拼接：导入图片后按顺序拼接，支持间距、边距、背景色、长图/切片导出。
- 批量替换：每个图片图层可添加替换素材，按替换序号批量导出。
- 分层套版：图片、文字、矩形作为图层自由摆放，支持替换导出。
- 一键套版：调用生图 API 生成套版效果，分屏模式下隐藏左侧工具栏。

## 分层工具栏
- 左侧竖排工具栏只在分层套版模式显示。
- 工具：
  - 移动工具：默认工具，可移动/缩放图层。
  - 文字工具：点击新增普通文字，框选新增固定范围自适应文字。
  - 矩形工具：拖拽绘制矩形图层。
- 按住 Ctrl 可在文字/矩形工具下临时使用移动功能，不影响滚轮缩放。
- 框选文字/矩形时实时显示范围，矩形框选使用接近最终矩形的预览样式。

## 文字图层
- 点击生成文字：按真实文字边界生成，后续修改文字也按真实边界收紧。
- 框选生成文字：图层边界保留为框选范围；文字在范围内等比缩放到最大并完整显示。
- 替换导出时，替换文字同样在框选范围内自动适配。
- 支持字体、字重、字号、对齐、填充、描边、描边粗细。
- 常用字体包含思源黑体、思源宋体、思源柔黑、思源真黑、阿里妈妈方圆体、阿里妈妈东方大楷、阿里妈妈数黑体等。
- 移动工具双击文字图层可直接编辑文字；文字工具单击文字图层可编辑文字。
- 文字替换面板第一行保留，第二行起有删除按钮。

## 矩形图层
- 拖拽生成矩形图层。
- 支持填充、描边、描边粗细、圆角。
- 矩形放大不使用旧图片拉伸，而是按当前尺寸重新绘制，避免模糊。
- 修改宽高时不锁定比例，圆角不随横纵拉伸变形。
- 矩形替换面板可填充图片，导出时按矩形范围等比充满裁切。

## 颜色与渐变
- 颜色面板支持纯色/渐变切换。
- 外层属性面板只显示色值输入，不显示透明度输入。
- 颜色弹窗里保留透明度输入框，默认 100。
- 渐变支持角度设置：
  - 圆形角度控件显示圆形边界、圆心和点击点。
  - 右侧有“渐变角度”数值输入框。
- 单个颜色拾色器内使用系统颜色选择器自身的透明度/取色能力。

## 更新发布
- 更新目录：`D:\软件更新服务`
- 清单地址：`http://192.192.3.180:8080/latest.json`
- 下载目录：`D:\软件更新服务\download`
- 发布脚本：`发布到局域网更新服务.bat`
- 当前打包方式：`npm run tauri build -- --no-bundle`
- 由于 WiX 安装包步骤曾失败，目前发布 exe 本体。
- 发布后清单包含：
  - `version`
  - `fileName` / `file_name`
  - `downloadUrl` / `url`
  - `notes`
- 软件启动自动检查更新，每小时自动检查一次，也可手动检查。
- 下载更新成功后自动打开下载文件夹；导出成功后自动打开目标文件夹。

## 常用验证命令
- `node --check src\main.js`
- `npm run build`
- `cargo check`（在 `src-tauri` 下）
- `cmd /c "发布到局域网更新服务.bat"`
- `curl.exe -s http://192.192.3.180:8080/latest.json`
- `curl.exe -I http://192.192.3.180:8080/download/designer-x.x.x.exe`

## 注意事项
- 修改功能后需要升版本号，否则旧版本不会收到更新提示。
- Tauri 命令如果新增，需要同步加入 `src-tauri/permissions/app.toml`。
- 不要误改 `Cargo.lock` 里第三方依赖版本，只改项目自身 `image_slicer_tauri` 版本。
